説明

・衝撃耐性/衝撃減衰

衝撃にどれだけ耐えられるかです。衝撃が耐性を上回った時、機体がよろめき行動が中断されます。
具体的には以下のように計算されます。
衝撃耐性290、衝撃減衰50の機体が衝撃250の攻撃を受けたとき
最初の攻撃 290-(250-50) = 90 → 0以上なのでよろめかない
二回目攻撃 90 -(250-50) = -110 → 0以下なのでよろめく
その後耐性が290に戻りリセットされる。一定時間でもリセットされる？ようです。

・通常速度/後退速度/ダッシュ速度

移動速度です。通常速度は攻撃しながらの移動、後退は攻撃しながら後ろに下がった時、ダッシュは攻撃せず移動のみ行った時の速度です。

・KB耐性

ノックバック耐性です。敵の攻撃を受けよろめいたとき、敵のノックバック力分押し込まれます。耐性があればその割合分移動距離が減少します。

・貫通耐性

脚部は専用機起動状態のみ使用されます。専用機起動すると胸部の貫通耐性が使用されず、武器と脚部の貫通耐性が使用される？ようです。
この機体に敵の攻撃が当たると貫通力がこの数値分消費されます。0以下になると貫通できません。
例えば、貫通力1.5の攻撃は貫通耐性0.6の敵を三体まで攻撃できます。
